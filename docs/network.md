# Netzwerk

Das Homelab läuft in einem privaten Heimnetz. Öffentliche Dokumentation verwendet bewusst anonymisierte Adressen.

## Grundsätze

- keine öffentlichen Portfreigaben als Teil dieses Portfolio-Auftrags
- keine Router- oder Cloudflare-Änderungen
- interne Dienste bleiben intern erreichbar
- Monitoring prüft interne Dienste und Systemzustände
- MAC-Adressen und vollständige IP-Pläne werden nicht veröffentlicht

## Zugriffskonzept

Administrative Zugriffe erfolgen kontrolliert über bestehende lokale Wege. Für die Portfolio-Dokumentation werden nur Rollen und Kommunikationsbeziehungen beschrieben, keine Zugangsdaten.
